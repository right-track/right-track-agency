Right Track Agency (Abstract)

node module: right-track-agency

This is an abstract Right Track Agency. This module provides the shared functionality between all of the Right Track Agencies.

Currently, it provides the ability to read and load the agency configuration properties.

Documentation

Documentation can be found in the /doc/ directory of this repository or online at https://docs.righttrack.io/right-track-agency.
