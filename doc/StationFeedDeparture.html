Station Feed Departure

Real-time and departure information about the departure of a single Trip from a Stop. Used by the StationFeed Class.

To access this Class directly:

const SF = require('right-track-agency/src/StationFeed');
const StationFeedDeparture = SF.StationFeedDeparture;
